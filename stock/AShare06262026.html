0 · 报告生成说明（需求与方法）

本报告如何生成 · 原始问题与口径确认

原始需求（用户问题，中文翻译）

如指令需要澄清，请先向我提问。使用 dailyA 获取中国A股数据并结合外部信息，我想了解过去5年的情况；每个问题都要给出理由，不要编造，始终使用可靠数据，对未知缺口请明确标出；分析时请采用"组合式推理 + 可靠数据"：
  • 按成交额（资金量）衡量，最活跃的个股有哪些；可实现的最大涨幅是多少
  • 它们属于哪些板块
  • 这些个股各板块的资金流，在 日 / 周 / 月 / 年 维度上是否存在规律
  • 这些板块中的龙头个股是谁（取前5名）
  • 输出分析摘要、明细与可视化，并以 AShare06262026.html 输出（中文文案与样式，浅色模式）

生成方式与可靠性

数据直接取自本地 PostgreSQL public.dailyA，由 SQL 聚合 6,345,966 行（5,811 只 × 1,211 个交易日，2021-06-28 至 2026-06-26）计算得到，未编造任何数据；所有可疑极值（如个股区间涨幅）均回溯原始价格校验。板块由"代码→公司→申万一级行业"映射，并对非显见代码联网核对（来源见页脚）；图表由 Chart.js 渲染。

已与用户确认的口径

  • "最活跃" = 5年累计成交额（dnvaltrd 求和）
  • "最大涨幅" = 区间内最低点至其后最高点的复权涨幅（peak-to-trough run-up，用复权价计算）
  • 板块 = 按个股联网归类（申万一级）；边界案例已标注
  • 每板块龙头取前5名（个股数不足5的板块按实际数量展示）
  • "近1年"报告：按近12个月（2025-06-27 至 2026-06-26）成交额重新选出最活跃前60，再按近1年最大涨幅每板块取前5
  • 未知 / 无法计算项（如"主力资金净流入"，源表无买卖方向数据）已明确标注为缺口，详见第3、7节

执行摘要

数据结论及其推理依据

  • 集中度：成交额前60的个股（约占5,811只的1%）每年吸纳全市场 13–15% 的成交额（前60成交额÷全市场成交额）。交易资金高度集中于少数大型、流动性强的个股。
  • 成交最活跃（5年累计成交额）：宁德时代(300750)、东方财富(300059)、中际旭创(300308)居前，其后为贵州茅台及光模块/AI产业链。
  • 最大涨幅：区间最大涨幅由光模块/AI硬件个股领跑——新易盛(300502) +8,220%、中际旭创(300308) +7,603%（自2022年10月低点）——并非成交最活跃的蓝筹。高成交≠高涨幅。
  • 资金轮动：成交额在电力设备与有色金属（2021–22新能源/锂电周期）见顶，随后明显转向电子、通信、计算机与传媒（2023→2026 AI/光模块/半导体周期）。通信板块成交额五年间增长约13倍。
  • 近一年：按近12个月成交额重新排名，AI存储/内存浪潮新进活跃榜——江波龙(301308)、香农芯创(300475)、佰维存储(688525)、德明利(001309)——与光模块个股并列。仅电子+通信即占近1年前60成交额约57%。
  • 日历效应弱：星期内效应很小（周一略高，约高于周均值3%）。真正的信号是逐年的板块轮动。

7 · 已知数据缺口与方法说明

1. 源表无行业字段——行业为外部映射（申万一级）并联网核对；边界案例见第2节。
2. 无资金净流入数据——仅有总成交额，故"资金流"=成交额，非主力净流入（第3节）。
3. 口径=成交额前60——板块汇总以龙头为主；"每板块前5"在该口径内（5年）/重新排名的近1年口径内（第5节）。
4. 部分年份不完整——2021自06-28起、2026至06-26止（各约半年，标注*）。
5. 以源数据为准——数值（含截至2026-06-26的价格）按库内存储原样呈现；已做内部一致性校验（价格×股本≈市值），未与第二数据源交叉核对。
6. 近1年口径——"龙头"按 2025-06-27 至 2026-06-26 区间最大涨幅排序；同时给出近1年买入持有收益作对比。
