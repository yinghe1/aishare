Power Tools

obsidian-cli and defuddle — AI that can act on your vault and read the web cleanly.

Reading vs. acting

The first four skills taught AI to write content in the right format. The power tools go further: they let AI act directly on a live Obsidian vault, and read from the web without the noise.

⚡

obsidian-cli

A command-line tool that talks directly to a running Obsidian. AI can read, create, search, append to, and manage notes — without you touching a file manually.

🧹

defuddle

A web-page extractor. Give it a URL — it strips navigation, ads, and clutter, returning just the article content as clean markdown. Saves tokens. Works everywhere.

💡
Why this changes what AI can do for you

Without these tools, AI can only write files you then have to save yourself. With obsidian-cli, AI can append to your daily note, search the vault, create notes silently in the background, and check what already exists — all in one conversation.

obsidian-cli — your vault's remote control

Think of the CLI as a remote control for Obsidian. Instead of clicking through menus, AI types commands that Obsidian executes immediately. Obsidian must be open for this to work.

obsidian read file="My Note" Read the full content of a note by name
obsidian create name="New Note" content="# Hello" silent Create a new note without opening it in Obsidian (silent = background)
obsidian append file="My Note" content="- [ ] New task" Add content to the end of an existing note — perfect for daily notes
obsidian search query="project alpha" limit=10 Search the vault and return matching notes (up to 10 results)
obsidian daily:append content="- Meeting at 3pm" Append directly to today's daily note without specifying the filename
obsidian property:set name="status" value="done" file="My Note" Update a frontmatter property without rewriting the whole note
obsidian tags sort=count counts List all tags in the vault, sorted by how often they are used
🔧
Plugin development too

The obsidian-cli skill also covers plugin development — reloading plugins, capturing errors, taking screenshots, and running JavaScript in the Obsidian context. Useful when building or debugging Obsidian plugins with AI assistance.

defuddle — the web, without the noise

Web pages are designed for humans: headers, footers, sidebars, cookie banners, ads, related articles. For AI reading a page, this noise wastes tokens and buries the actual content. Defuddle acts like a surgeon — it removes everything except the article body.

CODE 
# Basic usage — get clean markdown
defuddle parse <url> --md

# Save to a file in your vault
defuddle parse <url> --md -o content.md

# Get just the title
defuddle parse <url> -p title

# Get the description (for frontmatter)
defuddle parse <url> -p description
PLAIN ENGLISH

The --md flag outputs clean markdown (not raw HTML)

Fetch the URL and return just the article content as markdown

Same as above but write to a file — great for saving research to vault

-o sets the output filename

-p extracts a single metadata property instead of the full content

Useful for generating frontmatter: grab title, description, domain from the URL

WEBFETCH (raw)

Returns full HTML: navigation, ads, cookie banners, sidebar, footer, comments — thousands of tokens the AI must process before finding the article.

DEFUDDLE (clean)

Returns only the article body as clean markdown. 10-20x fewer tokens. AI reads faster, costs less, and focuses on what matters.

End-to-end: save a web article to your vault

Here is how defuddle and obsidian-cli work together to turn a URL into a properly formatted vault note in one go.

1
You share a URL

"Save this article to my vault: https://example.com/great-article"

2
AI runs defuddle

defuddle parse <url> --md — strips clutter, returns clean markdown content. Also grabs title and description with -p flags.

3
AI composes the note

Uses obsidian-markdown skill to wrap content in frontmatter (title, source URL, date, tags) and format any callouts or highlights.

4
AI creates the vault note

obsidian create name="Article Title" content="..." silent — the note appears in your vault immediately, no manual saving needed.
