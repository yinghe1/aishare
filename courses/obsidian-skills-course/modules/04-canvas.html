A canvas is stored as a JSON file with the .canvas extension. The json-canvas skill teaches AI to generate valid JSON that Obsidian can render as a visual board — without you needing to write a single line of JSON yourself.

What a canvas file actually contains

Think of a canvas file like a stage direction script for a play. It says: "Place Actor A at position X, facing Y. Draw a line from Actor A to Actor B." Obsidian reads the script and renders the stage.

CODE 
{
  "nodes": [
    {
      "id": "6f0ad84f44ce9c17",
      "type": "text",
      "x": 0, "y": 0,
      "width": 400, "height": 200,
      "text": "# Hello World"
    }
  ],
  "edges": [
    {
      "id": "0123456789abcdef",
      "fromNode": "6f0ad84f44ce9c17",
      "toNode": "a1b2c3d4e5f67890",
      "label": "leads to"
    }
  ]
}
PLAIN ENGLISH

The entire canvas is one JSON object with two lists

"nodes" is the list of things on the board

Each node is its own object with properties

A unique 16-character ID — no two nodes can share this

"text" means it holds markdown text (other types: file, link, group)

Position on the infinite canvas — x goes right, y goes down

How big the box is in pixels

The markdown content displayed inside the box

"edges" is the list of connecting lines between nodes

Each edge also gets a unique ID

fromNode: the ID of the node where the arrow starts

toNode: the ID of the node where the arrow points

Optional text label that appears on the connecting line

Four types of canvas card

Every item on a canvas is a node. The type property controls what the node displays.

📝

text

A plain text or markdown card. Write headers, lists, callouts — anything markdown supports. Most common type.

📄

file

Displays an existing note from your vault as a preview card. Clicking opens the note. Can show a specific heading with subpath.

🌐

link

Embeds an external URL — shows a web page preview card on the canvas. Great for research boards.

📦

group

A container that visually groups other nodes. Like drawing a box around related cards. Has a label and optional background.

⚠️
Every ID must be unique — and the skill enforces this

The biggest canvas bug is two nodes with the same ID. The skill instructs AI to generate fresh 16-character hex IDs for every node and edge, and to verify that fromNode/toNode references actually exist in the nodes list before writing the file.
